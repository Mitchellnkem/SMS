Getting Started

Welcome

Genius I.t Brainery Management System is a powerful user-friendly system designed as per user's perspective with multiple MIS reports / logins.

The number of school is multiplying faster. Parents have become more attentive about their children, their activities and progress in the school. With our School Management System, parents will receive complete information of their child their Attendance, Daily Time table, Progress reports (current as well as previous) from the comfort of their home.SMS '&' Email facility ensures the parents to remain updated with school activities. All these things required a lot of man power if done manually and it is a bulky system too. Now we provide you a system which is process centric '&' continues its functionalities with lesser number of staff. Now no tension about employee job changing and finding datas. Everything is now electronically preserved.

INSTALLATION GUIDE

  • FOR LOCAL INSTALLATION OR SETUP
  • extract the downloaded folder,
  • go to databse
  • open the sql file and set it up in local server
  • go to config folder
  • click on the config folder
  • open database.php
  • change the settings to your database settings

  • FOR SERVER INSTALLATION
  • extract the folder to your root directory
  • set up the datbase
  • import the database to your server
  • go to application
  • click on config folder
  • open database.php
  • change the settings to your own database settings
